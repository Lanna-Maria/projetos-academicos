# 🗄️ Banco de Dados – BD1

> Atividade prática desenvolvida utilizando o banco **Empresa**, com foco em consultas SQL e subconsultas envolvendo `ALL`, `ANY`, `MAX`, `MIN`, `AVG`, `JOIN` e filtros condicionais.

---

## 📚 Contexto da Atividade

O objetivo da atividade é treinar o uso de consultas SQL relacionadas a:

✔ Funções de agregação  
✔ Subconsultas com `ALL` e `ANY`  
✔ Filtros condicionais  
✔ Operadores lógicos  
✔ Relações entre tabelas (`JOIN`)  
✔ Uso de `MIN`, `MAX`, `AVG`  
✔ Consultas por categoria, fornecedor e funcionário  

Todas as consultas executam operações sobre o banco **Empresa** (modelo didático comum em cursos de Banco de Dados).

---

## 🧩 Exercícios Resolvidos

A atividade contém oito consultas, resumidas abaixo:

1. **Produtos com maior preço** (ordenados por nome)  
2. **Pedido com maior frete em 1996**  
3. **Funcionários do Brasil com salário acima da média**  
4. **Produtos da categoria Bebidas com menor preço**  
5. **Produtos de Bebidas que aparecem em outra categoria**  
6. **Produtos de um fornecedor com preço superior ao de outro**  
7. **Funcionários sem pedidos no ano de 1999**  
8. **Produtos de Confeitos com preço superior aos de Condimentos**

---

## 🧠 Conceitos Reforçados

📌 Subconsultas (`ALL`, `ANY`)  
📌 Funções agregadas (`MAX`, `MIN`, `AVG`)  
📌 Filtragem por ano (`YEAR`)  
📌 `JOIN` entre tabelas  
📌 Operadores de comparação  
📌 `NOT IN` & filtragem negativa  

---

## 📝 Pré-requisitos

Para execução dos scripts é necessário um SGBD compatível com SQL padrão, como:

- SQL Server *(recomendado para esse modelo de banco)*
- PostgreSQL
- MySQL / MariaDB  
*(podendo exigir pequenas adaptações)*
