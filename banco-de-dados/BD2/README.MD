# 🗄️ Banco de Dados – BD2

> Exercícios, consultas SQL, revisões e exemplos práticos desenvolvidos na disciplina de **Banco de Dados II**.

---

## 📋 Conteúdo da Pasta

Esta pasta contém arquivos **SQL** com consultas e exercícios relacionados aos seguintes tópicos:

✔ Consultas simples  
✔ Consultas com `SELECT`  
✔ `JOIN` e subconsultas  
✔ Operações `INSERT`, `UPDATE` e `DELETE`  
✔ Revisão de `DML`  
✔ `HAVING`, `GROUP BY` e funções de agregação  
✔ `VIEW` e abstração de dados
