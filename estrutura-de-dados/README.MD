# 🌳 Estrutura de Dados

> Atividades, algoritmos e implementações desenvolvidas na disciplina de **Estrutura de Dados**, durante o curso de **Bacharelado em Ciência da Computação – IFCE Tianguá**.

---

## 🎓 Sobre a Disciplina

A disciplina de Estrutura de Dados visa apresentar as principais estruturas utilizadas para armazenamento, organização e manipulação de dados em memória, desenvolvendo raciocínio computacional eficiente e compreensão de complexidade.

Os conteúdos envolvem:

✔ Alocação estática e dinâmica  
✔ Ponteiros e manipulação de memória  
✔ Arrays e vetores  
✔ Listas lineares (estáticas e encadeadas)  
✔ Pilhas  
✔ Filas *(dependendo do semestre)*  
✔ Estruturas compostas (structs)  
✔ Operações de inserção, remoção, busca e iteração  

---

## 📁 Subpastas

> Vetores dinâmicos, soma de valores e manipulação de strings via alocação.

🔹 [`listaEstatica`](./listaEstatica)

🔹 [`struct`](./struct)  
> Criação de estruturas compostas para armazenamento de múltiplos tipos.
